# 《寿元劫》技能系统详细设计（完善版）

> 无职业限制，自由搭配。格挡弹反机制创造硬核战斗体验。所有玩家都是输出，根据功法创造不同流派。

---

## 一、技能系统总览

### 1.1 核心理念

**无职业分类**：
- 所有玩家都是输出，没有坦克、治疗、输出之分
- 根据主修功法和灵根属性，自由选择技能
- 创造独特的输出流派

**格挡弹反机制**：
- 每个玩家必须学习1-2个格挡技能
- 用于抵挡BOSS的必杀技能
- 时机把握是关键，完美格挡触发反击

**技能搭配自由**：
- 技能槽位系统，根据境界解锁
- 主动技能、被动技能、格挡技能自由搭配
- 五行相克，创造克制策略
