# 《寿元劫》新手引导流程设计

> 从出生到进入修仙界,0-2小时完整体验

---

## 一、新手引导总览

### 核心理念

**循序渐进,寓教于乐**

- 通过剧情引导,自然学习游戏机制
- 每个系统都有对应的教学任务
- 新手期节奏紧凑,体验丰富
- 目标明确,让玩家知道下一步要做什么

### 新手期时长

**总时长**：0-2小时(游戏时间)

**阶段划分**：

| 阶段 | 时长 | 核心内容 |
|------|------|---------|
| **阶段1：出生与觉醒** | 0-10分钟 | 创建角色、了解背景、基础移动 |
| **阶段2：技能学习** | 10-40分钟 | 学习采集、钓鱼、轻功、挖矿 |
| **阶段3：村外历练** | 40-70分钟 | 战斗教学、格挡弹反、完成历练 |
| **阶段4：山贼危机** | 70-100分钟 | 第一个副本、团队配合、主线触发 |
| **阶段5：前往天元城** | 100-120分钟 | 商队之旅、进入修仙界 |
