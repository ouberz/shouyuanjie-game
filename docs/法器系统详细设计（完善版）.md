# 《寿元劫》法器系统详细设计（完善版）

> 法器者，修士之助也。或防御、或辅助、或特殊，皆为战斗之利器。法器不同于武器，更注重功能性而非单纯伤害。

---

## 一、法器系统总览

### 1.1 法器定位

**法器与武器的区别**：

| 维度 | 武器 | 法器 |
|------|------|------|
| **主要功能** | 直接伤害输出 | 辅助、防御、特殊功能 |
| **伤害占比** | 总伤害40-60% | 总伤害10-30%或无伤害 |
| **装备位置** | 主手/副手 | 法器槽（独立槽位） |
| **技能类型** | 攻击技能为主 | 辅助、防御、特殊技能为主 |
| **冷却时间** | 较短（1-5秒） | 较长（10-300秒） |
| **灵气消耗** | 低（10-50点） | 高（50-300点） |
