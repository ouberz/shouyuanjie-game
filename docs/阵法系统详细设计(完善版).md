## 一、阵法系统总览

阵法系统是《寿元劫》的核心玩法之一，玩家可以通过布置阵法获得强大的战斗辅助能力。阵法系统与**五行系统深度结合**，通过五行相生相克创造丰富的策略组合。

### 阵法分类

**按功能分类**：
- **防御阵法**：护盾、结界、反弹伤害
- **攻击阵法**：单体/群体伤害、困杀阵
- **辅助阵法**：治疗、增益、控制
- **特殊阵法**：传送、隐匿、封印、修炼加速

**按规模分类**：
- **小型阵法**（1-5人）：单人/小队使用
- **中型阵法**（6-25人）：团队副本/门派战
- **大型阵法**（26-100人）：世界BOSS/战场
- **超大型阵法**（100+人）：门派驻地防御/护国大阵

## 二、五行阵法原理

### 2.1 五行相生系统

**相生关系**：金→水→木→火→土→金

| 主阵法 | 相生阵法 | 效果加成 |
|--------|---------|---------|
| 金阵 | 水阵 | 水阵效果+50% |
| 水阵 | 木阵 | 木阵效果+50% |
| 木阵 | 火阵 | 火阵效果+50% |
| 火阵 | 土阵 | 土阵效果+50% |

## 五、阵法布置与维护

### 布置流程

1. 学习阵法秘籍
2. 准备布置材料（灵石、阵旗、精华等）
3. 选择布置地点
4. 按照阵法图案放置阵眼
5. 激活阵法
6. 维持阵法（小型阵法可自动维持）

### 维护周期

- 小型阵法：无需维护
- 中型阵法：每月维护一次
- 大型阵法：每周维护一次
- 超大型阵法：每月维护一次

## 七、阵法创造系统（高级玩法）

### 创造条件

- **阵法师等级**：宗师级（5级）以上
- **境界要求**：金丹期以上
- **功法要求**：五行功法大成
- **前置任务**：完成《阵法大师》成就

### 创造流程

1. 前往【天机阁】或【阵法师公会】
2. 提交创造申请（消耗贡献点×500）
3. 完成资格认证任务
4. 通过【阵法师宗师】认证考核
5. 获得【阵法创造资格】

### 创造要素

- 阵眼布局（数量、位置、能量流向）
- 阵纹绘制（类型、密度、走向）
- 五行配置（主属性、辅助属性、比例）
- 效果设计（主动/被动/特殊效果）
- 能量系统（来源、消耗、持续时间）

## 八、阵法系统总结

### 核心设计理念

1. **五行深度结合**：五行相生相克创造策略性
2. **多人协作**：大型阵法需要团队配合
3. **创造自由**：宗师级阵法师可自创阵法
4. **成长系统**：阵法师等级与境界挂钩
5. **实用性**：防御/攻击/辅助/特殊四大类型满足不同需求

### 关键特色

- 五行相生相克机制
- 多人协作布置阵法
- 阵法师职业成长
- 阵法创造系统
- 丰富的阵法类型
