# 战斗系统详细设计（完善版）

## 一、战斗系统概述

战斗是《寿元劫》的核心玩法。游戏采用**无职业全员DPS**设计，所有玩家都可输出、自保、辅助。战斗强调**操作技巧**而非数值碾压，格挡弹反机制是核心特色。

### 战斗系统核心理念

- **操作为王**：格挡、弹反、闪避、反击等技巧决定胜负
- **全员输出**：无坦克/治疗分工，人人都是DPS
- **属性平衡**：五行相克、境界压制、装备搭配影响战斗
- **策略多样**：技能组合、功法搭配、阵法运用、法器使用

## 二、战斗基础机制

### 2.1 战斗属性系统

#### 基础属性

| 属性名称 | 属性说明 | 影响内容 |
|---------|---------|----------|
| 生命值 | 玩家的血量 | 归零则死亡 |
| 灵力值 | 使用技能的资源 | 技能消耗灵力 |
| 体力值 | 物理攻击的资源 | 武器攻击消耗体力 |
| 攻击力 | 基础伤害 | 所有伤害的基础 |
| 防御力 | 减少受到的伤害 | 物理防御/法术防御 |
| 速度 | 移动速度、攻击速度 | 影响战斗节奏 |
| 暴击率 | 暴击触发概率 | 暴击造成1.5倍伤害 |
| 暴击伤害 | 暴击时的伤害倍率 | 基础150% |

#### 五行属性

| 五行属性 | 属性效果 | 相克关系 |
|---------|---------|----------|
| 金属性 | 攻击力+10% | 克木，被火克 |
| 木属性 | 生命恢复+20% | 克土，被金克 |
| 水属性 | 灵力恢复+20% | 克火，被土克 |
| 火属性 | 暴击伤害+20% | 克金，被水克 |
| 土属性 | 防御力+15% | 克水，被木克 |
| 雷属性 | 攻速+15%（特殊属性） | 不参与相克 |

**五行相克规则**：
- 攻击克制属性目标：伤害+20%
- 攻击被克属性目标：伤害-20%

### 2.2 攻击类型

#### 武器攻击（物理伤害）

| 攻击方式 | 消耗 | 基础伤害 | 攻击速度 |
|---------|------|---------|----------|
| 轻攻击 | 体力5 | 100% | 快 |
| 重攻击 | 体力15 | 180% | 慢 |
| 连击 | 体力10/次 | 80%×3 | 中 |
| 蓄力攻击 | 体力30 | 300% | 很慢 |

#### 法术攻击（法术伤害）

| 攻击方式 | 消耗 | 基础伤害 | 施法时间 |
|---------|------|---------|----------|
| 瞬发法术 | 灵力20-50 | 80%-150% | 瞬发 |
| 吟唱法术 | 灵力50-100 | 200%-400% | 1-3秒 |
| 引导法术 | 灵力10/秒 | 50%/秒 | 引导 |
| 范围法术 | 灵力80-150 | 120%-250% | 1-2秒 |

### 2.3 伤害计算公式

#### 基础伤害公式

```
最终伤害 = (基础攻击 × 技能倍率 - 目标防御) × 五行修正 × 其他修正
```

#### 伤害修正系数

| 修正类型 | 修正系数 | 说明 |
|---------|---------|------|
| 暴击 | ×1.5 - ×2.0 | 暴击伤害属性 |
| 五行克制 | ×1.2 | 克制目标 |
| 五行被克 | ×0.8 | 被目标克制 |
| 境界压制 | ×1.1/阶 | 高一境界 |
| 破防状态 | ×1.5 | 目标破防 |
| 格挡成功 | ×0.3 | 目标格挡 |
| 弹反成功 | ×0 | 目标弹反（无伤害） |

## 三、格挡与弹反系统（核心特色）

### 3.1 格挡系统

#### 格挡基础机制

| 格挡属性 | 格挡效果 |
|---------|----------|
| 格挡触发 | 按住格挡键，进入格挡状态 |
| 格挡消耗 | 体力10/秒（持续格挡） |
| 格挡减伤 | 减少70%伤害 |
| 格挡方向 | 前/左/右/上/下五方向 |
| 完美格挡 | 伤害判定前0.3秒内格挡，减伤90%，不消耗体力 |
| 格挡失败 | 方向错误或时机错误，承受全额伤害 |

#### 格挡方向判定

| 攻击类型 | 正确格挡方向 | 错误格挡方向 |
|---------|-------------|-------------|
| 正斩 | 前 | 左/右/上/下 |
| 左斩 | 左 | 前/右/上/下 |
| 右斩 | 右 | 前/左/上/下 |
| 上挑 | 上 | 前/左/右/下 |
| 下劈 | 下 | 前/左/右/上 |

#### 格挡状态

| 格挡状态 | 触发条件 | 格挡效果 |
|---------|---------|----------|
| 普通格挡 | 按住格挡键 | 减伤70%，消耗体力 |
| 完美格挡 | 伤害判定前0.3秒格挡 | 减伤90%，不消耗体力，触发反击窗口 |
| 连续格挡 | 短时间内多次格挡 | 格挡效果递减（70%→60%→50%→40%） |
| 破盾 | 体力耗尽时格挡 | 格挡失效，进入破防状态 |

### 3.2 弹反系统

#### 弹反基础机制

| 弹反属性 | 弹反效果 |
|---------|----------|
| 弹反触发 | 伤害判定前0.2秒内格挡 |

| 弹反窗口 | 成功弹反后有1.5秒反击时间 |

|---------|---------|---------|----------|
| 轻攻击 | 0.25秒 | 简单 | 反击伤害+30% |

| 失衡状态 | 触发条件 | 失衡效果 | 持续时间 |
|---------|---------|----------|----------|

| 完全失衡 | 连续弹反/弹反BOSS技能 | 倒地状态，受到伤害+50% | 5秒 |

## 八、战斗系统总结

### 核心战斗循环

1. **装备武器/法器** → 确定战斗风格
2. **配置技能** → 搭配技能连招
3. **遭遇敌人** → 分析敌人攻击模式
4. **格挡/弹反/闪避** → 防御敌人攻击
5. **反击/连招** → 输出伤害
6. **使用道具** → 补充状态
7. **击杀敌人** → 获得奖励

### 战斗系统特色

- **操作导向**：格挡弹反机制让操作技巧决定胜负
- **全员输出**：无职业分工，人人都是DPS
- **连招系统**：技能连招、武器连招、空中连招多种组合
- **五行相克**：五行属性影响战斗伤害
- **PVP/PVE平衡**：竞技场、野外PVP、副本BOSS各有特色
- **成长系统**：熟练度、成就系统让战斗持续成长

### 战斗技巧要点

- **学会观察**：观察敌人攻击前摇，判断攻击类型
- **精准格挡**：方向正确、时机精准是格挡关键
- **敢于弹反**：高风险高回报，弹反是反杀核心
- **灵活闪避**：无法格挡的攻击及时闪避
- **连招熟练**：熟练掌握连招，输出最大化
- **道具使用**：关键时刻使用道具扭转战局
