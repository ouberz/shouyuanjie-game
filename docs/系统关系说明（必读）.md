# 《寿元劫》系统关系说明（必读）

> 本文档明确各系统的定义、边界和关系，避免重复设计。

---

## 一、核心系统架构

### 1.1 系统层级关系

```
玩家角色
├─ 基础属性
│   ├─ 境界（修仙等级）
│   ├─ 寿元（核心资源）
│   └─ 灵根（五行属性）
│
├─ 修炼系统
│   ├─ 功法（决定修炼方向和速度）
│   │   ├─ 主修功法（核心心法）
│   │   ├─ 辅修功法（辅助加成）
│   │   └─ 神通技能（功法附带的技能）
│   │
│   └─ 技能（战斗中的实际操作）
│       ├─ 格挡技能（防御机制）
│       ├─ 主动技能（攻击技能）
│       └─ 被动技能（永久加成）
│
├─ 装备系统
│   ├─ 武器（伤害输出）
│   ├─ 法器（辅助增益）
│   ├─ 防具（防御减伤）
│   └─ 饰品（特殊效果）
│
└─ 特殊系统
    ├─ 阵法（多人协作）
    ├─ 飞行（移动方式）
    └─ 丹药（消耗品）
```

## 二、功法系统 vs 技能系统（重点区分）

### 2.1 功法系统定义

**核心功能**：决定修炼方向、成长速度、属性加成

**主要内容**：
- 主修功法：修炼心法，决定修炼速度和主属性
- 辅修功法：提供额外属性加成
- 功法修炼等级：需要时间积累，提升效果
- 神通技能：功法达到一定等级后自动解锁的技能

**关键特点**：
- 功法是被动加成，永久生效
- 功法修炼需要长时间（几十到几百小时）
- 功法决定修炼速度（灵气吸收效率）
- 功法品质影响渡劫成功率

**示例**：
```
《太白金经》（主修功法）
- 品质：上品（蓝色）
- 修炼等级：大成
- 效果：
  - 攻击力+30%（永久被动）
  - 破甲+20%（永久被动）
  - 修炼速度+40%（被动加成）
  - 神通技能：金刃术（功法附带技能，小成解锁）
  - 神通技能：金剑阵（功法附带技能，大成解锁）
```

### 2.2 技能系统定义

**核心功能**：战斗中的实际操作，主动使用

**主要内容**：
- 格挡技能：防御机制，需要手动触发
- 主动技能：攻击技能，有冷却时间
- 被动技能：永久生效的属性加成
- 技能熟练度：使用次数提升，增强效果

**关键特点**：
- 技能是主动使用，有冷却时间
- 技能熟练度提升相对快速（几百次使用）
- 技能决定战斗风格
- 技能可自由搭配（技能槽位系统）

**示例**：
```
《金刃术》（主动技能）
- 来源：功法《太白金经》小成解锁
- 熟练度：精通（使用了500次）
- 效果：发射金属性剑气，造成攻击力×250%伤害
- 冷却时间：3秒
- 消耗：灵气15点
- 熟练度加成：
  - 熟练：伤害×220%
  - 精通：伤害×250%，穿透+20%
  - 大师：伤害×280%，穿透+40%，必中
```

### 2.3 两者关系

**功法提供技能**：
- 功法修炼到一定等级，自动解锁对应神通技能
- 解锁后的技能进入技能库，可自由装备
- 技能强度与功法修炼等级相关

**技能独立存在**：
- 技能也可通过其他途径获得（副本、BOSS、奇遇）
- 同一技能可能有多个来源
- 技能熟练度独立于功法修炼等级

**设计原则**：
- 功法是"内功"，决定基础属性
- 技能是"招式"，决定战斗操作
- 功法修炼等级高 → 解锁更多技能
- 技能熟练度高 → 技能效果更强

## 三、设计原则总结

### 3.1 避免重复

**规则**：
1. 功法提供被动加成和少量神通技能
2. 技能是主动操作，可自由搭配
3. 武器提供基础伤害和武器技能
4. 法器提供被动效果
5. 阵法是多人协作的战略技能
6. 飞行是移动系统，独立存在
7. 丹药是消耗品，提供临时效果

### 3.2 系统协作

**原则**：
1. 各系统效果可以叠加
2. 玩家可自由组合系统
3. 系统间有明确边界
4. 核心体验：功法决定方向，技能决定操作，装备决定数值
