# 《寿元劫》功法系统详细设计（完善版）

> 万法归宗，道法自然。功法是修士的根本，决定修炼方向、战斗风格、成长速度。

---

## 一、功法体系总览

### 1.1 功法分类体系

```
功法体系
├─ 凡人功法（0-2小时）
│   ├─ 生活技能（采集、钓鱼、挖矿、烹饪、锻造）
│   └─ 基础武学（轻功、拳脚、兵器、内功）
│
├─ 修仙功法（炼气期-元婴期）
│   ├─ 主修功法（核心修炼心法，决定修炼方向）
│   │   ├─ 五行主修功法（金木水火土）
│   │   ├─ 特殊主修功法（雷、阴阳、空间、时间）
│   │   └─ 门派主修功法（各门派特色功法）
│   │
│   ├─ 辅修功法（辅助修炼、增加属性）
│   │   ├─ 属性增幅类（攻击、防御、速度）
│   │   ├─ 修炼加速类（灵气吸收、感悟提升）
│   │   ├─ 寿元增加类（延寿、驻颜）
│   │   └─ 特殊能力类（神识、遁术、隐匿）
│   │
│   └─ 秘术神通（战斗技能、特殊能力）
│       ├─ 攻击型神通（单体、范围、持续）
│       ├─ 防御型神通（护盾、减伤、闪避）
│       ├─ 控制型神通（眩晕、冰冻、封印）
│       └─ 辅助型神通（加速、治疗、增益）
│
└─ 传承功法（元婴期以上）
    ├─ 上古传承（上古修士遗留）
    ├─ 天道传承（天道宗独家）
    └─ 妖族传承（妖族专属）
```

### 1.2 功法品质等级

**品质等级**（由低到高）：

| 品质 | 颜色 | 获取难度 | 威力倍率 | 修炼速度 | 突破难度 |
|------|------|---------|---------|---------|---------|
| **凡品** | 灰色 | 极低（新手赠送） | 基础×1.0 | 基础 | 无 |
| **下品** | 白色 | 低（商店购买） | 基础×1.2 | +10% | 低 |
| **中品** | 绿色 | 中低（任务奖励） | 基础×1.5 | +20% | 中低 |
| **上品** | 蓝色 | 中（副本掉落） | 基础×2.0 | +30% | 中 |
| **极品** | 紫色 | 中高（BOSS掉落） | 基础×2.5 | +40% | 高 |
| **天品** | 橙色 | 高（上古传承） | 基础×3.0 | +50% | 极高 |
| **仙品** | 金色 | 极高（传说） | 基础×4.0 | +60% | 传说 |

**设计意图**：
- 品质越高，功法威力越强，但获取和突破难度也越高
- 让玩家有明确的追求目标

### 1.3 功法修炼等级

**修炼等级**（由低到高）：

| 修炼等级 | 境界要求 | 效果增幅 | 所需时间 | 突破条件 |
|---------|---------|---------|---------|---------|
| **入门** | 无要求 | 100% | 无 | 无 |
| **初窥** | 炼气初期 | 110% | 5小时 | 悟性丹×1 |
| **小成** | 炼气中期 | 125% | 15小时 | 悟性丹×2 + 感悟石×1 |
| **登堂** | 炼气后期 | 145% | 30小时 | 悟性丹×3 + 感悟石×3 |
| **大成** | 筑基初期 | 170% | 60小时 | 悟性丹×5 + 感悟石×5 + 灵乳×1 |
| **圆熟** | 筑基中期 | 200% | 100小时 | 悟性丹×10 + 感悟石×10 + 灵乳×3 |
| **圆满** | 筑基后期 | 235% | 150小时 | 突破丹×1 + 感悟石×20 + 灵乳×5 |
| **化境** | 金丹初期 | 275% | 250小时 | 突破丹×3 + 天材地宝×1 |
| **归真** | 金丹中期 | 320% | 400小时 | 突破丹×5 + 天材地宝×3 + 奇遇×1 |
| **入道** | 金丹后期 | 370% | 600小时 | 突破丹×10 + 天材地宝×5 + 奇遇×3 |
| **合道** | 元婴初期 | 420% | 1000小时 | 传承×1 + 奇遇×5 |
| **得道** | 元婴中期 | 480% | 2000小时 | 传承×3 + 天道感悟×1 |

**设计意图**：
- 功法修炼等级需要长时间积累，让玩家有持续追求
- 高等级功法需要稀有资源，增加资源获取动力
- 传承和奇遇作为高等级突破条件，增加玩法多样性

## 二、凡人功法系统（0-2小时）

### 2.1 生活技能

#### 采集技能

**技能名称**：《基础采药术》

**技能等级**：
- 入门：可采集1级草药（春华草、铁线草）
- 初窥：可采集2级草药（炎阳花、月华菇）
- 小成：可采集3级草药（灵参、仙芝）
- 登堂：可采集4级草药（天材地宝）

**采集速度**：
- 入门：10秒/株
- 初窥：8秒/株
- 小成：6秒/株
- 登堂：4秒/株

**采集成功率**：
- 入门：70%
- 初窥：80%
- 小成：90%
- 登堂：100%

**采集产出**：
- 成功：获得草药×1-3
- 失败：获得草药碎片×1（可合成）

**学习条件**：无（新手村NPC教学）

**升级方式**：
- 采集次数积累
- 使用技能书升级

---

#### 钓鱼技能

**技能名称**：《基础钓鱼术》

**钓鱼等级**：
- 入门：可钓普通鱼类（鲤鱼、鲫鱼）
- 初窥：可钓稀有鱼类（金线鱼、银鳞鱼）
- 小成：可钓灵鱼（灵鲤、仙鱼）
- 登堂：可钓妖鱼（蛟鱼、龙鱼）

**钓鱼机制**：

**时间机制**：
- 白天：普通鱼类刷新率高
- 夜晚：稀有鱼类刷新率高
- 特定时间：灵鱼刷新（凌晨、黄昏）

**天气影响**：
- 晴天：普通鱼类活跃
- 雨天：稀有鱼类活跃
- 雷雨：妖鱼出现概率高

**钓鱼小游戏**：
1. 抛竿：选择钓点
2. 等待：观察浮漂
3. 提竿：在正确时机按下空格键
4. 搏斗：控制力度条，保持在中段

**成功奖励**：
- 完美提竿：鱼×2 + 钓鱼熟练度×2
- 普通提竿：鱼×1 + 钓鱼熟练度×1
- 失败：无奖励

**学习条件**：无（新手村NPC教学）

---

#### 挖矿技能

**技能名称**：《基础采矿术》

**矿石等级**：
- 入门：铁矿、铜矿
- 初窥：银矿、金矿
- 小成：灵石矿、玄铁矿
- 登堂：天材矿、仙矿

**挖矿机制**：

**矿脉刷新**：
- 普通矿脉：每小时刷新
- 稀有矿脉：每6小时刷新
- 灵石矿脉：每12小时刷新
- 天材矿脉：每24小时刷新

**挖矿工具**：
| 工具名称 | 耐久度 | 采矿速度 | 成功率 | 价格 |
|---------|--------|---------|--------|------|
| 铁镐 | 50 | 基础 | 70% | 10灵石 |
| 精铁镐 | 100 | +20% | 80% | 50灵石 |
| 灵铁镐 | 200 | +40% | 90% | 200灵石 |
| 天铁镐 | 500 | +60% | 100% | 1000灵石 |

**挖矿小游戏**：
1. 发现矿脉
2. 选择采矿工具
3. 按节奏点击鼠标（跟随节拍）
4. 完美节奏：矿石×2
5. 普通节奏：矿石×1
6. 失败节奏：耐久度消耗×2

**学习条件**：无（新手村NPC教学）

### 2.2 基础武学

#### 轻功

**技能名称**：《基础轻功》

**效果**：
- 移动速度+10%
- 可跳跃障碍物
- 可攀爬低矮建筑

**修炼等级**：
- 入门：移动速度+10%，跳跃高度+10%
- 初窥：移动速度+15%，跳跃高度+20%
- 小成：移动速度+20%，跳跃高度+30%
- 登堂：移动速度+25%，跳跃高度+40%

**学习条件**：无（新手村NPC教学）

**进阶轻功**：
- 《踏雪无痕》：移动速度+30%，可在水面行走
- 《梯云纵》：可三段跳跃
- 《凌波微步》：闪避率+20%

---

#### 拳脚功夫

**技能名称**：《基础拳脚》

**效果**：
- 攻击力+5%
- 解锁基础拳脚技能

**技能列表**：
| 技能名称 | 效果 | 冷却时间 | 消耗 |
|---------|------|---------|------|
| **直拳** | 造成基础攻击×1.0伤害 | 1秒 | 无 |
| **勾拳** | 造成基础攻击×1.2伤害，概率眩晕1秒 | 3秒 | 体力5 |
| **连击** | 连续攻击3次，总伤害×0.8 | 5秒 | 体力10 |
| **重击** | 造成基础攻击×1.5伤害，概率击退 | 4秒 | 体力8 |

**修炼等级**：
- 入门：攻击力+5%，解锁直拳
- 初窥：攻击力+10%，解锁勾拳
- 小成：攻击力+15%，解锁连击
- 登堂：攻击力+20%，解锁重击

**学习条件**：无（新手村NPC教学）

### 2.3 凡人功法学习流程

**0-30分钟**：
- 出生，与村长对话
- 学习《基础采药术》
- 学习《基础钓鱼术》
- 学习《基础采矿术》

**30-60分钟**：
- 完成采集任务（采集草药×10）
- 完成钓鱼任务（钓鱼×5）
- 完成挖矿任务（挖矿×5）

**60-90分钟**：
- 学习《基础轻功》
- 学习《基础拳脚》
- 学习《基础兵器》
- 选择武器类型（剑/刀/枪/棍/鞭/扇）

**90-120分钟**：
- 完成战斗教学（击杀野怪×10）
- 完成轻功教学（跳跃障碍×5）
- 触发主线剧情：发现修仙者战斗
- 选择：帮助/旁观/逃跑
- 剧情分支：获得修仙者指引
- 前往修仙界入口
- 突破炼气期

## 九、总结

### 9.1 核心设计理念

1. **循序渐进**：凡人功法→修仙功法→传承功法，成长路线清晰
2. **五行相克**：简单易懂的五行系统，策略性强
3. **品质驱动**：品质越高，威力越强，获取和突破难度也越高
4. **修炼养成**：功法等级需要时间积累，让玩家有持续追求
5. **组合策略**：功法组合效果，鼓励玩家收集和搭配

### 9.2 关键亮点

- **六大属性主修功法**：金木水火土雷，各有特色
- **妖族专属功法**：雕族、龟族、蛟族，独特的修炼体系
- **特殊主修功法**：血神经、鬼修夺魂诀，邪道特色
- **功法组合系统**：五行组合、门派组合，策略性强
- **修炼养成**：长时间积累，稀有资源突破，持续追求
