# 《寿元劫》门派系统设计

> 门派是修士成长的摇篮,功法、资源、社交的核心载体

---

## 一、门派系统总览

### 核心理念

**门派是修士的第二故乡**

- 提供功法、装备、丹药支持
- 创造社交环境(师兄弟、长老、掌门)
- 规范修士行为(门规、因果)
- 不同门派有不同特色和玩法

### 门派分类

```
门派类型
├─ 正道门派
│   ├─ 天道宗(综合型,最强势力)
│   ├─ 丹霞谷(炼丹型,延寿丹垄断)
│   └─ 剑宗(战斗型,战斗力最强)
│
├─ 邪道门派
│   ├─ 血神教(夺舍型,公开夺舍)
│   └─ 鬼修门(神魂型,专研神魂之术)
│
└─ 中立势力
    ├─ 散修联盟(松散型,资源共享)
    └─ 商会(商业型,不参与争斗)
```
