# 飞行系统详细设计（完善版）

## 一、飞行系统概述

飞行系统是修仙者脱离凡胎、遨游天地的重要标志。玩家需要达到**筑基期**才能解锁飞行能力。飞行不仅是移动方式，更是战斗、探索、社交的核心玩法。

### 飞行系统核心机制

- **境界门槛**：筑基期解锁基础飞行，境界越高飞行速度越快、高度越高
- **飞行工具**：飞剑、法器、灵兽、符箓等多种飞行工具
- **灵力消耗**：飞行持续消耗灵力，灵力耗尽强制降落
- **空战系统**：飞行状态下可进行空中战斗
- **种族差异**：雕族天生飞行优势，龟族/蛟族飞行能力较弱

### 2.2 飞行工具属性

每件飞行工具具有以下属性：

| 属性 | 说明 |
|------|------|
| **基础速度** | 飞行工具的基础移动速度 |
| **灵力效率** | 飞行时灵力消耗倍率 |
| **稳定性** | 飞行时受攻击/气流影响的抗干扰能力 |
| **加速度** | 从静止到最高速的时间 |
| **最大高度** | 飞行工具可达到的最高飞行高度 |
| **载重量** | 可承载的重量上限（影响速度） |
| **耐久度** | 飞行工具的耐久，归零损坏 |
| **特殊效果** | 独特的飞行特效（隐身、加速、护盾等） |

### 2.3 飞行工具强化

#### （1）锻造强化

### 4.2 飞行限制

#### （1）境界限制

- **凡人期**：无法飞行
- **炼气期**：可使用符箓飞行，持续时间短
- **筑基期**：解锁飞行，但有高度/时间限制
- **金丹期**：自由飞行，高度大幅提升
- **元婴期及以上**：无限制

#### （2）区域限制

| 区域类型 | 飞行限制 | 原因 |
|---------|---------|------|
| **新手村** | 禁止飞行 | 保护新手 |
| **城池内部** | 禁止飞行/限高50米 | 城市秩序 |
| **门派禁地** | 禁止飞行 | 门派规矩 |
| **风暴海域** | 金丹期以下无法飞越 | 恶劣天气 |
| **妖族海域** | 金丹期以下进入会被击落 | 妖族领地 |
| **仙境秘境** | 部分区域禁飞 | 秘境规则 |
| **战场副本** | 部分区域禁飞 | 战术平衡 |

#### （3）灵力限制

| 飞行状态 | 灵力消耗（每秒） |
|---------|----------------|
| **悬停** | 1点灵力 |
| **正常飞行** | 2点灵力 |

| **冲刺** | 10点灵力 |

## 五、空战系统

### 5.1 空战基础机制

#### （1）空战触发条件

- 空中遇到敌对玩家/NPC
- 主动攻击空中目标
- 护送任务遭遇袭击
- 门派战争空战
- 恶人谷无差别PVP

#### （2）空战特色机制

| 机制名称 | 机制说明 |
|---------|----------|
| **高度优势** | 高空攻击低空目标，伤害+10%；低空攻击高空，伤害-10% |
| **速度优势** | 高速状态下攻击力+10%，但命中率-10% |
| **俯冲攻击** | 俯冲攻击伤害+50%，但自身受击伤害+20% |
| **空中闪避** | 空中闪避消耗灵力，成功闪避后可瞬移反击 |
| **坠机惩罚** | 空中被击落，生命值归零或受到重伤，坠落伤害 |
| **空中格挡** | 成功格挡可触发"反击风暴"，造成范围伤害 |

#### （3）空战操作

| 操作 | 按键（默认） | 效果 |
|------|------------|------|
| **升空** | Space | 垂直升空 |
| **俯冲** | Ctrl + W | 快速下降 |
| **冲刺** | Shift | 短距离高速移动 |
| **转向** | A/D | 水平转向 |
| **翻滚** | Q/E | 闪避翻滚 |
| **空中攻击** | 鼠标左键 | 空中攻击 |
| **空中格挡** | 鼠标右键 | 空中格挡 |
| **空中弹反** | 右键（完美时机） | 弹反成功后反击 |

## 九、飞行系统总结

### 核心玩法循环

1. **境界提升** → 解锁飞行能力
2. **获取飞行工具** → 飞剑/法器/灵兽/符箓
3. **学习飞行功法** → 提升飞行性能
4. **强化飞行工具** → 锻造/符文/器灵
5. **练习空战技巧** → 格挡/弹反/冲刺/俯冲
6. **参与空战事件** → 护送/侦查/竞速/空战
7. **获取成就称号** → 属性加成/展示身份

### 飞行系统特色

- **多样性**：四种飞行工具类型，满足不同玩家需求
- **成长性**：飞行能力随境界提升，工具可强化
- **战术性**：空战包含高度/速度/俯冲/格挡等策略
- **种族差异**：雕族速度、龟族防御、蛟族海空双域
- **社交性**：多人飞行、护送任务、团队空战
- **探索性**：高空探索、奇遇系统、隐藏仙境
