# 《寿元劫》- 寿元即生命

> 一款以寿元为核心的古典仙侠游戏，五行聚齐渡劫系统，格挡弹反战斗机制。

## 🎮 游戏简介

《寿元劫》是一款古典仙侠大世风格的修仙游戏。核心特色：

- **寿元即生命**：在线时间 = 寿元消耗，创造紧迫感
- **五行聚齐渡劫**：灵根纯净度的双刃剑设计
- **格挡弹反战斗**：无职业全员DPS，硬核战斗机制
- **传承系统**：死亡不是结束，而是新的开始
- **陆海双界**：陆地人族主场，海洋妖族主场

## 🌟 核心特色

### 寿元系统
- 寿元是核心资源，在线时间持续消耗
- 境界提升增加寿元上限
- 延寿丹药、功法、奇遇可延长寿元
- 寿元归零 = 角色死亡（可传承或夺舍）

### 五行渡劫
- 灵根纯净度影响修炼速度
- 渡劫需要五行聚齐
- 纯净灵根修士渡劫更难
- 创造资源交换和社交需求

### 战斗系统
- 无职业分类，全员DPS
- 格挡弹反机制（完美格挡窗口0.3秒）
- 技能自由搭配
- 五行相克策略

### 传承系统
- 寿元将尽时建造洞府
- 设置传承物品和防御机制
- 新角色可寻找传承
- 驻颜丹传承容貌
